Listed below are a few featured projects I have completed. Feel free to click on any to get some more information.
C# .NET Bug Tracker App

.NET web app that allows developers to register and login and provides an enivronment to create, view, edit, assign and close tickets for their project. The app provides a clean UI for developers to view ticket statistics, search for specific tickets, developer (all & user specific) assignments and all created tickets (open & closed).
*Demo video included on GitHub*

Tech stack: C#, ASP.NET MVC, SQL, RAZOR, Bootstrap

Angular Weather App

Angular web app that displays the local time (updates every second) and requires the user to authenticate themselves by signing in via a google account in order for them to get access to view the current weather.

Tech stack: Angular, OpenWeatherMap API, Travis CI, TypeScript, JavaScript, Firebase, HTML5, CSS3

React Full Stack Job Tracker App

React app that allows a user to create, delete and set reminders for created jobs/tasks. This app implements a mock back end (JSON-server REST API) to allow storage/manipulation of the data on a backend so user can leave and return to the app as they please without any loss of data.

Tech stack: React JS, JSON-server REST API

Java Gym Membership Management App

Java program that allows a user to manage current list of registered members while allowing the user to add, delete and flag members that have missed payments. Also allows the user to enter in monthly revenue and expenses to create a simple income statement for each month of the current year.

Tech stack: Java, Java Swing GUI toolkit
